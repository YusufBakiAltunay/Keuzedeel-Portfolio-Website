# 📄 Portfolio Website – Voorstel

## 📱 Devices (Apparaten)

Deze portfolio website wordt responsive gemaakt voor de volgende apparaten:

### 🔹 Mobiel (portrait / staand)
- Smartphones (iPhone / Android)  
- Layout: 1 kolom  
- Focus op eenvoudige navigatie en grote knoppen  

### 🔹 Mobiel (landscape / liggend)
- Smartphones horizontaal gebruikt  
- Layout: 2 kolommen waar mogelijk  

### 🔹 Desktop / Laptop
- Grote schermen  
- Layout: meerdere kolommen (grid)

---

## 🌐 Browser ondersteuning

De website wordt ondersteund op:

- Google Chrome (laatste 2 versies)

---

## 📏 Resoluties (Schermgroottes)

De volgende breakpoints worden gebruikt:

| Device type        | Breedte (px)        | Layout |
|--------------------|---------------------|--------|
| Mobiel (portrait)  | 320px – 480px       | 1 kolom |
| Mobiel (landscape) | 481px – 768px       | 2 kolommen |
| Desktop            | 769px – 1440px+     | 3+ kolommen |

---

## ♿ Toegankelijkheid (WCAG A)

De website voldoet minimaal aan WCAG A richtlijnen:

- Afbeeldingen bevatten alt-tekst  
- Voldoende kleurcontrast  
- Correct gebruik van labels bij formulieren  
- Duidelijke heading-structuur (h1, h2, h3)  
- Klikbare elementen zijn duidelijk herkenbaar
